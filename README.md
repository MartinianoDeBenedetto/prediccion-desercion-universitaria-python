# prediccion-desercion-universitaria-python
Modelo de Machine Learning para predecir el riesgo de deserción y egreso de estudiantes. Trabajo Final de Prácticas Pre-Profesionales (UGR)

Este proyecto fue desarrollado durante mis **Prácticas Pre-Profesionales** en la **Universidad del Gran Rosario (UGR)**. El objetivo fue aplicar Ciencia de Datos para identificar alumnos en riesgo de abandono y predecir la probabilidad de egreso.

El modelo permite a la institución detectar de forma temprana a estudiantes que podrían necesitar apoyo académico o administrativo, basándose en su rendimiento y situación socio-laboral.

## Aspectos Técnicos
* **Modelado:** Utilicé el algoritmo **Random Forest** (Bosques Aleatorios) por su gran capacidad para manejar variables categóricas y numéricas.
* **Rendimiento:** El modelo alcanzó un **ROC AUC de 0.92**, demostrando una excelente capacidad predictiva.
* **Variables Clave:** Se identificó que el promedio académico y la situación laboral son los factores de mayor peso en la permanencia.

## Herramientas
* **Python** (Pandas, Scikit-Learn, Matplotlib, Seaborn).
* **Análisis de Datos:** Limpieza de bases institucionales y normalización de variables.

> **Nota de Privacidad:** Los datasets originales (CSV/Excel) no se incluyen en este repositorio para proteger la información confidencial de los estudiantes de la UGR. El código se presenta con los outputs de datos sensibles limpios.
